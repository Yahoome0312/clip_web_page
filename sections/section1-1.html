1.1 Standard Supervised Learning

For nearly a decade, progress in computer vision was driven by a single dominant paradigm: Standard Supervised Learning. Models like ResNet, VGG, and AlexNet were trained to answer one specific question:

"Which of these $N$ predefined categories does this image belong to?"

The workflow follows a strict formula: $$ \text{Input Image} \rightarrow \text{Extract Features} \rightarrow \text{Predict Fixed Class Label} $$

The Closed World Assumption

This paradigm operates in a "closed world". It assumes a fixed label set (ontology) and a fixed decision head. Once trained, the model's vocabulary is frozen. If a model is trained on ImageNet (which has 1,000 classes), its entire universe consists of only those 1,000 concepts.

The Core Flaw: Ontological Rigidity

This approach suffers from a fundamental limitation called Ontological RigidityThe inability of a system to adapt its knowledge structure (ontology) to new concepts without re-training..

1. Fixed Vocabulary

The model can only recognize concepts it was explicitly taught. It cannot recognize "COVID-19 Face Masks" or a "Tesla Cybertruck" because these concepts didn't exist in the ImageNet dataset (created ~2012).

2. Loss of Semantics

To the model, Class #10 (Dalmatian) and Class #15 (German Shepherd) are just orthogonal indices in a One-Hot vector: $$ [0, ..., 1, ..., 0] $$ The mathematical distance between two dog breeds is the same as the distance between a "Dog" and a "Taxicab". The model loses the semantic hierarchy that exists in the real world.
